class TextDecoder

package js.node.util

@:jsRequire("util","TextDecoder")

An implementation of the WHATWG Encoding Standard TextDecoder API.

Constructor

new (?encoding:String, ?options:TextDecoderOptions)

Creates an new TextDecoder instance.

The encoding may specify one of the supported encodings or an alias.

Variables

read onlyencoding:String

The encoding supported by the TextDecoder instance.

read onlyfatal:Bool

The value will be true if decoding errors result in a TypeError being thrown.

read onlyignoreBOM:Bool

The value will be true if the decoding result will include the byte order mark.

Methods

decode (?input:ArrayBuffer, ?options:TextDecodeOptions):String

decode (?input:ArrayBufferView, ?options:TextDecodeOptions):String

Decodes the input and returns a string. If options.stream is true, any incomplete byte sequences occurring at the end of the input are buffered internally and emitted after the next call to decode().

If fatal is true, decoding errors that occur will result in a TypeError being thrown.
